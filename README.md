# Caregiver Calling Agent (Deployable)

This repository contains a deployable Flask-based AI calling system:
- Twilio integration for automated calls & recordings
- Telegram notifications
- Weekly scraper for hospitals & elderly homes (sample entries included)
- Transcription & GPT summaries (OpenAI)
- Simple mobile-friendly dashboard (suitable for iPhone)
- Basic learning module to prioritize hospitals over time

**Important:** Fill your secrets into `.env` (use `.env.example` as a template). Do NOT commit secrets.

Deploy on Render, Railway, or DigitalOcean following the instructions in the project description.
